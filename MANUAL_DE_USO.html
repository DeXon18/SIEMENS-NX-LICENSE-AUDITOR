MANUAL DE USO: SIEMENS NX LICENSE AUDITOR

Este documento explica cómo utilizar la herramienta SIEMENS NX LICENSE AUDITOR (DX-CORE) para gestionar las licencias de sus productos Siemens (NX y Designcenter).

1. Introducción

Esta herramienta permite cambiar rápida y seguramente entre dos tipos de licencias:

  • Licencia Local: Servidor tradicional de la empresa (ej: 29000@192.168.1.150).
  • Licencia Cloud: Nuevas licencias basadas en la nube.

✅ El programa protege automáticamente las versiones antiguas que no soportan la nube para que no dejen de funcionar.

2. Primeros Pasos

Ejecución

  1. Localice la carpeta con los archivos descargados.
  2. Haga doble clic en el archivo Switch-License.bat.
  3. Windows pedirá permisos de Administrador. Haga clic en Sí.

⚠ Nota: Los permisos de administrador son obligatorios para poder cambiar la configuración del sistema.

Configuración Inicial (Solo la primera vez)

Al abrir el programa por primera vez, le pedirá el servidor de licencias:

  • Escriba la dirección (ej: 29000@miservidor) y presione ENTER.

3. Pantalla Principal (Dashboard)

Verá una tabla con el estado actual de sus programas:

ESTADO SIGNIFICADO
[ OK ] Configurado correctamente en Local.
[ NUBE ] Configurado para usar licencia Cloud.
[ PROTEGIDO ] Versión antigua bloqueada en Local por seguridad.

4. Opciones Disponibles

Use los números del teclado para elegir una opción:

[1] Cambiar a NUBE (Cloud)

Cambia todas las versiones compatibles a modo nube.
(Las versiones antiguas NO se tocarán).

[2] Cambiar a LOCAL

Restaura todo al servidor de licencias de la empresa.

[3] Modificar Servidor Local

Úselo si cambia la IP del servidor de licencias.

5. Seguridad y Logs

⚠ Advertencia de Programas Abiertos

Si intenta cambiar la licencia con NX o DesignCenter abierto, el programa le avisará en color AMARILLO.

Cierre los programas antes de continuar para evitar errores.

Historial

Se genera un archivo audit.log donde queda registrado quién hizo el cambio y cuándo.
